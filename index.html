There are a large number of Cloud storage products available in the market today. Popular services include Google Drive, Dropbox, Box.net, Amazon Cloud drive and Microsoft Skydrive to name a few. However, while a wide range of these services have free storage, an user will not have a great way to consolidate his storage across this services, avoid duplicating packages and store more files than the free storage limits. My Big Fat Box is the solution to that.

My Big Fat Box addresses the problem by creating a list of all the storage service providers, the user has signed up for, and maintains the files in multiple underlying storage, while keeping the meta-data about where the files are really stored. From an user's perspective, he/she just sees a much bigger box! Also, duplicates can now be avoided across the different services.
